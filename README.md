## 项目概述
本项目为前后端分离架构。其中front包为前端，hrmapp包为后端。

__前后端使用绝对地址的方式写入页面标签，使用的时候注意换成自己设定的端口__


### front
编写方式为java+javabean+前端三件套。大范围的使用了frameset用于单一页面的分区

### hrmapp
借用的已有的后端管理平台并进行了本地化和定制化。
包中有原始sql，不是本项目数据库sql

__拦截器和视图解析器存在问题__
